# 💾 Cache Kurulum Rehberi

#### 🔰 Hikayenin başlamasına az kaldı!
[Oyun Kurulumu](https://ucp.fast-rp.com/wiki/installing-game) sekmesinde işin bitti mi? GTA:V ve FiveM'i kurduysan sırada cache var diye düşünüyor olabilirsin ama hayır, Fast'te böyle bir zorunluluğun yok.

Sunucuya giriş yaptığın anda gerekli olan, düşük boyutlu birkaç modül dosyalarını yüklersin ve koskoca cache dosyasının inmesini beklemeden oyuna dahil olabilirsin! İndirmen gereken dosyalar, sen oyunun tadını çıkarırken sana fark ettirmeden arka planda yavaş yavaş iniyor olacak.
